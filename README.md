# AI-Powered SEO Content Generation System

## 📌 Project Overview
This project demonstrates how AI and prompt engineering can be used to generate SEO-optimized blog content and content clusters for local businesses.

The system is designed to replicate how digital marketing agencies plan and execute SEO strategies.

## 🎯 Objective
- Generate SEO-friendly blog content
- Build content clusters (pillar + supporting blogs)
- Target local SEO keywords
- Improve search engine ranking potential

## 🏪 Business Use Case
Digital Marketing Agency in Kochi

Target audience:
- Small businesses
- Startups
- Local service providers

## 🧠 Features
- Keyword + search intent generation
- SEO blog outline creation (H1–H3)
- Long-form blog generation
- Supporting blog generation
- Internal linking strategy
- Local SEO optimization

## 📂 Project Structure
- prompts/ → reusable AI prompts
- content/ → generated blogs
- strategy/ → SEO planning and linking

## 🚀 How It Works
1. Input business type and location
2. Generate keywords and intent
3. Create pillar blog outline
4. Generate long-form blog
5. Create supporting blogs
6. Link content for SEO ranking

## 📈 Outcome
This system helps businesses:
- Rank on Google
- Build authority
- Generate leads through content

## 💡 Key Insight
This project shows how structured prompt engineering can replace manual SEO content planning.
