# SEO Content System

An AI-powered SEO content generation system for creating keyword-optimized blog content targeting the Kochi digital marketing market.

## Folder Structure

```
seo-content-system/
│
├── README.md
├── prompts/
│   ├── keyword-generator.txt       # Prompt to generate SEO keywords
│   ├── pillar-outline.txt          # Prompt to outline a pillar blog post
│   ├── blog-writer.txt             # Prompt to write a full pillar blog
│   ├── supporting-blog.txt         # Prompt to write supporting/cluster blogs
│   ├── internal-linking.txt        # Prompt to suggest internal links
│
├── content/
│   ├── pillar-blog.md              # Main pillar blog post
│   ├── seo-services-kochi.md       # Supporting blog: SEO Services in Kochi
│   ├── social-media-kochi.md       # Supporting blog: Social Media in Kochi
│   ├── google-ads-kochi.md         # Supporting blog: Google Ads in Kochi
│   ├── choosing-agency-kochi.md    # Supporting blog: Choosing an Agency in Kochi
│
├── strategy/
│   ├── keyword-research.md         # Keyword research strategy and findings
│   ├── internal-linking.md         # Internal linking strategy
```

## How to Use

1. **Keyword Research** – Use `prompts/keyword-generator.txt` with an AI tool to generate targeted SEO keywords.
2. **Plan Content** – Use `prompts/pillar-outline.txt` to create an outline for the pillar blog.
3. **Write Pillar Blog** – Use `prompts/blog-writer.txt` to generate the full pillar blog post.
4. **Write Supporting Blogs** – Use `prompts/supporting-blog.txt` for each cluster/supporting article.
5. **Link Internally** – Use `prompts/internal-linking.txt` to generate internal linking suggestions.
6. **Review Strategy** – Refer to `strategy/` for keyword and linking documentation.
